Sistema de Mensajería  
Desafío de CSS Avanzado – Desarrollo Web

Este proyecto consiste en la creación de una interfaz de mensajería tipo chat, aplicando principios de *CSS Avanzado*, *Flexbox*, *layouts responsivos*.

El objetivo principal es replicar una estructura típica de una aplicación de mensajería moderna, con una columna de contactos y una vista central informativa.
